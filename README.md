# SimpleList
Simple List Android App\
SimpleList is a simple check list application\

Features:\
    Auto Save List On Exit\
    Auto Load List On Start\
    Save Option, Save Any Time\
    Remove All\
    Remove Checked\
    Clear All Check Marks\
